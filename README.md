<h3>구현기능🔥</h3>
✅읽기, 쓰기, 조회의 화면으로 구성<br>
✅작성한 닉네임은 다음번 게시물 작성시 자동입력<br>
✅게시물에대해 삭제가능<br>
✅게시물작성시 중요게시물은 체크가능<br>
✅중요게시물은 list 배경의 색깔이 다르게나타남<br>
✅데이터 베이스 구현X(로컬스토리지 사용함)<br>
✅타이틀과 닉네임의 검색기능<br>
